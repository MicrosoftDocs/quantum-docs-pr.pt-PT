## <a name="representing-two-qubit-states"></a><span data-ttu-id="d6f75-135">Representar estados de dois qubits</span><span class="sxs-lookup"><span data-stu-id="d6f75-135">Representing two-qubit states</span></span>

<span data-ttu-id="d6f75-136">Nos exemplos anteriores, o estado de um qubit foi descrito com uma matriz de coluna única $\begin{bmatrix} a \\\\  b \end{bmatrix}$ e a aplicação de uma operação ao mesmo foi descrita ao multiplicar as duas matrizes.</span><span class="sxs-lookup"><span data-stu-id="d6f75-136">In the examples above, the state of one qubit was described using a single column matrix $\begin{bmatrix} a \\\\  b \end{bmatrix}$, and applying an operation to it was described by multiplying the two matrices.</span></span> <span data-ttu-id="d6f75-137">No entanto, os computadores quânticos utilizam mais de um qubit. Então, como é que descrevemos o estado combinado de dois qubits?</span><span class="sxs-lookup"><span data-stu-id="d6f75-137">However, quantum computers use more than one qubit, so how do you describe the combined state of two qubits?</span></span> 

<span data-ttu-id="d6f75-138">Lembre-se de que cada qubit é um espaço de vetor, pelo que não podem ser simplesmente multiplicados.</span><span class="sxs-lookup"><span data-stu-id="d6f75-138">Remember that each qubit is a vector space, so they can't just be multiplied.</span></span> <span data-ttu-id="d6f75-139">Em vez disso, vai utilizar um **produto tensorial**, que é uma operação associada que cria um espaço de vetor novo a partir de espaços de vetores individuais e é representado pelo símbolo $\otimes$.</span><span class="sxs-lookup"><span data-stu-id="d6f75-139">Instead, you use a **tensor product**, which is a related operation that creates a new vector space from individual vector spaces, and is represented by the $\otimes$ symbol.</span></span> <span data-ttu-id="d6f75-140">Por exemplo, o produto tensorial do estado de dois qubits $\begin{bmatrix} a \\\\  b \end{bmatrix}$ e $\begin{bmatrix} c \\\\  d \end{bmatrix}$ é calculado</span><span class="sxs-lookup"><span data-stu-id="d6f75-140">For example, the tensor product of two qubit states $\begin{bmatrix} a \\\\  b \end{bmatrix}$ and $\begin{bmatrix} c \\\\  d \end{bmatrix}$ is calculated</span></span>

<span data-ttu-id="d6f75-141">$$ \begin{bmatrix} a \\\\  b \end{bmatrix} \otimes \begin{bmatrix} c \\\\  d \end{bmatrix} =\begin{bmatrix} a \begin{bmatrix} c \\\\  d \end{bmatrix} \\\\ b \begin{bmatrix}c \\\\  d \end{bmatrix} \end{bmatrix} = \begin{bmatrix} ac \\\\  ad \\\\  bc \\\\  bd \end{bmatrix}.</span><span class="sxs-lookup"><span data-stu-id="d6f75-141">$$ \begin{bmatrix} a \\\\  b \end{bmatrix} \otimes \begin{bmatrix} c \\\\  d \end{bmatrix} =\begin{bmatrix} a \begin{bmatrix} c \\\\  d \end{bmatrix} \\\\ b \begin{bmatrix}c \\\\  d \end{bmatrix} \end{bmatrix} = \begin{bmatrix} ac \\\\  ad \\\\  bc \\\\  bd \end{bmatrix}.</span></span>
$$

<span data-ttu-id="d6f75-142">O resultado é uma matriz quadridimensional, em que cada elemento representa uma probabilidade.</span><span class="sxs-lookup"><span data-stu-id="d6f75-142">The result is a four-dimensional matrix, with each element representing a probability.</span></span> <span data-ttu-id="d6f75-143">Por exemplo, $ac$ é a probabilidade de ambos os qubits colapsarem para 0 e 0, $ad$ é a probabilidade de 0 e 1, e assim sucessivamente.</span><span class="sxs-lookup"><span data-stu-id="d6f75-143">For example, $ac$ is the probability of the two qubits collapsing to 0 and 0, $ad$ is the probability of 0 and 1, and so on.</span></span> 

<span data-ttu-id="d6f75-144">Tal como um estado de qubit único $\begin{bmatrix} a \\\\  b \end{bmatrix}$ tem de cumprir o requisito em que $|a|^2 + |b|^2 = 1$ de modo a representar um estado quântico, um estado de dois qubits $\begin{bmatrix} ac \\\\  ad \\\\  bc \\\\  bd \end{bmatrix}$ tem de cumprir o requisito em que $|a|^2 + |b|^2 + |c|^2+ |d|^2 = 1$.</span><span class="sxs-lookup"><span data-stu-id="d6f75-144">Just as a single qubit state $\begin{bmatrix} a \\\\  b \end{bmatrix}$ must meet the requirement that $|a|^2 + |b|^2 = 1$ in order to represent a quantum state, a two-qubit state $\begin{bmatrix} ac \\\\  ad \\\\  bc \\\\  bd \end{bmatrix}$ must meet the requirement that $|a|^2 + |b|^2 + |c|^2+ |d|^2 = 1$.</span></span>
