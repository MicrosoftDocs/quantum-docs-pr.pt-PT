# <a name="readme"></a>README
**IMPORTANTE:** para que as imagens se tornem corretamente em modo escuro, deve evitar transparências.
- Para ficheiros .jpg não precisa de fazer nada, uma vez que o formato de ficheiro não suporta elementos transparentes.
- Para ficheiros .png deve adicionar um fundo branco ou alterar o valor do canal alfa para 100. A forma mais fácil de o fazer no Windows é abrindo o ficheiro em Paint e economizando, sobrewritting o ficheiro original.
- Para ficheiros .svg deve adicionar um retângulo branco na camada mais baixa. Pode fazê-lo com o Inkscape:
  - Abra o ficheiro .svg.
  - Selecione a ferramenta de máquina de fazer quadrados e desenhe um retângulo branco em cima da figura original.
  - Selecione a ferramenta "Selecione e transforme objetos" clicando na seta escura ou pressionando F1.
  - Enquanto seleciona o retângulo, clique no elemento da barra de ferramentas "Seleção inferior a baixo (Fim)".
  - Ajuste o retângulo com o rato ou as teclas de seta.
