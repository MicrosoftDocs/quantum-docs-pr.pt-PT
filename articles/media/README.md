# <a name="readme"></a>README
**IMPORTANTE:** para que as imagens sejam corretamente renderizadas em modo escuro, deve evitar transparências.
- Para ficheiros .jpg não precisa de fazer nada, uma vez que o formato de ficheiro não suporta elementos transparentes.
- Para ficheiros .png deve adicionar um fundo branco ou alterar o valor do canal alfa para 100. A maneira mais fácil de o fazer no Windows é abrindo o ficheiro em Paint e economizando, sobretorcendo o ficheiro original.
- Para ficheiros .svg deve adicionar um retângulo branco na camada mais baixa. Pode fazer isto com a Inkscape:
  - Abra o ficheiro .svg.
  - Selecione a ferramenta de fabricante de quadrados e desenhe um retângulo branco em cima da figura original.
  - Selecione a ferramenta "Selecione e transforme objetos" clicando na seta escura ou pressionando F1.
  - Ao ter o retângulo selecionado, clique no elemento da barra de ferramentas "Seleção inferior para baixo (Fim)".
  - Ajuste o retângulo com o rato ou as teclas de seta.
