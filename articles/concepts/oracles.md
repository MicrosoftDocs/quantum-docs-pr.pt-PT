<span data-ttu-id="fcf17-185">Para tal, considere uma entrada binária específica $ x = (x_ { 0 } , x_ { 1 , } \pontos, x_ { n-1 } ) $ .</span><span class="sxs-lookup"><span data-stu-id="fcf17-185">To do so, consider a particular binary input $x = (x_{0}, x_{1}, \dots, x_{n-1})$.</span></span>

## <a name="defining-an-oracle-by-its-effect-on-computational-basis-states"></a><span data-ttu-id="fcf17-191">Definição de um oráculo pelo seu efeito em estados de base computacional</span><span class="sxs-lookup"><span data-stu-id="fcf17-191">Defining an oracle by its effect on computational basis states</span></span>

$$
\begin{align}
    <span data-ttu-id="fcf17-194">O( \ket { x } \otimes \ket { y } ) x = \ket { } \otimes \ket { y \oplus f(x) } .</span><span class="sxs-lookup"><span data-stu-id="fcf17-194">O(\ket{x} \otimes \ket{y}) = \ket{x} \otimes \ket{y \oplus f(x)}.</span></span>
