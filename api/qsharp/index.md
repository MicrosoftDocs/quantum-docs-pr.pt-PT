# <a name="q-libraries"></a>Bibliotecas Q# #

A biblioteca padrão Q# inclui operações, funções e tipos definidos pelo utilizador utilizados para desenvolver programas quânticos. Também estão incluídas três bibliotecas adicionais criadas com base na biblioteca padrão: química, números e machine learning.

Para obter mais informações sobre bibliotecas Q#, veja <xref:microsoft.quantum.libraries>.
