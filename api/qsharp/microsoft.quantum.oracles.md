# <a name="microsoftquantumoracles-namespace"></a>Espaço de nome Microsoft.Quantum.Oracles

Este espaço de nome fornece tipos definidos pelo utilizador para decorar vários tipos de oráculos pelo seu uso pretendido em algoritmos quânticos, tais como estimativa de fase e amplificação de amplitude.
