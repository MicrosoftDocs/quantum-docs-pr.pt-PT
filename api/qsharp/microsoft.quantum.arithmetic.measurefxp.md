# <a name="measurefxp-operation"></a>Operação MeasureFxP

Espaço de nome: [Microsoft.Quantum.Aritmética](xref:Microsoft.Quantum.Arithmetic)

Pacote: [Microsoft.Quantum.Nummerics](https://nuget.org/packages/Microsoft.Quantum.Numerics)


Meça um número de ponto fixo, devolva o seu valor como Double, e repõe todo o registo para zero.

```qsharp
operation MeasureFxP (fp : Microsoft.Quantum.Arithmetic.FixedPoint) : Double
```


## <a name="input"></a>Entrada

### <a name="fp--fixedpoint"></a>FP : [Ponto Fixo](xref:Microsoft.Quantum.Arithmetic.FixedPoint)

Número de ponto fixo para medir.



## <a name="output--double"></a>Saída : [Duplo](xref:microsoft.quantum.lang-ref.double)
