# <a name="addconstantfxp-operation"></a>Operação AddConstantFxP

Espaço de nome: [Microsoft.Quantum.Aritmética](xref:Microsoft.Quantum.Arithmetic)

Pacote: [Microsoft.Quantum.Nummerics](https://nuget.org/packages/Microsoft.Quantum.Numerics)


Adiciona uma constante clássica a um número de ponto fixo quântico.

```qsharp
operation AddConstantFxP (constant : Double, fp : Microsoft.Quantum.Arithmetic.FixedPoint) : Unit is Adj + Ctl
```


## <a name="input"></a>Entrada

### <a name="constant--double"></a>constante : [Duplo](xref:microsoft.quantum.lang-ref.double)

Constante para adicionar ao número de ponto fixo quântico.


### <a name="fp--fixedpoint"></a>FP : [Ponto Fixo](xref:Microsoft.Quantum.Arithmetic.FixedPoint)

Número de ponto fixo ao qual a constante será adicionada.



## <a name="output--unit"></a>Saída : [Unidade](xref:microsoft.quantum.lang-ref.unit)
