# <a name="complexpolarascomplex-function"></a>Função ComplexPolarAsComplex

Espaço de nome: [Microsoft.Quantum.Math](xref:Microsoft.Quantum.Math)

Pacote: [](https://nuget.org/packages/)


Converte um número complexo de tipos `ComplexPolar` para um número complexo de `Complex` tipos.

```qsharp
function ComplexPolarAsComplex (input : Microsoft.Quantum.Math.ComplexPolar) : Microsoft.Quantum.Math.Complex
```


## <a name="input"></a>Entrada

### <a name="input--complexpolar"></a>entrada : [ComplexPolar](xref:Microsoft.Quantum.Math.ComplexPolar)

Número complexo $c = r e^{i t}$.



## <a name="output--complex"></a>Saída : [Complexo](xref:Microsoft.Quantum.Math.Complex)

Número complexo $c = x + i y$.
